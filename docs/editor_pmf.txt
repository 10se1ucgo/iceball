PMF editing guide

Use pkg/iceball/pmfedit.

Default keys:

i/k = move up/down
j/l = move left/right
u/o = move forward/back

a/d = rotate around y axis
w/s = rotate around x axis
q/e = rotate around y2 axis

+/- = enlarge/shrink block
g = add/paint block
b = remove block


F3 = quickload clsave/vol/editor.pmf
F10 = quicksave clsave/vol/editor.pmf
